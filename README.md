# Mission Control - Spixi Mini App

A dashboard for Home Assistant and MQTT management, built as a Spixi Mini App.

## Features
- Home Assistant Entity Explorer
- Real-time MQTT state updates
- Customizable Grid Layout
- Spixi SDK integration for bidirectional messaging

## Installation
In Spixi, use the following URL to install the appinfo:
`https://raw.githubusercontent.com/subsubl/mission-control/main/appinfo.spixi`

## Development
```bash
npm install
npm run build
```
The build artifacts are moved to the `app/` folder for Spixi compatibility.
